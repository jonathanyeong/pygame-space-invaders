# Space Invaders 

> PyGame clone of Space Invaders

# Usage

To run the app:

1. `$ python pyvader/space_invader.py`

To test the app:

1. Make sure you are in the directory above tests. Which is the same directory
as your README.md.
2. Run `$ nosetests`
